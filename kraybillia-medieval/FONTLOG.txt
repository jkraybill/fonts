FONTLOG for Kraybillia Medieval
===============================

This file provides detailed information on the Kraybillia Medieval font
software. This information should be distributed along with
the Kraybillia Medieval fonts and any derivative works.

Basic Font Information
----------------------
Kraybillia Medieval is a hand-daubed display family: every glyph is
drawn by a seeded parametric pen, with contextual alternates
cycling multiple independent renders of each letter. Full
Google Fonts Latin Core coverage (EN/RO/VI at the heart,
plus CE, Baltic, Turkish, Welsh, Icelandic).
