# Kraybillia Fonts

Hand-built display typefaces by JK (Jeremy Kraybill) + Gordo —
the public release side of a private parametric foundry.

**Status: pre-release.** Binaries, specimens, and the release
gallery land here once the foundry cuts each version. Licensed
under the SIL Open Font License 1.1 ("Kraybillia" is a Reserved
Font Name).

## Families

- **Kraybillia Old** — daub-pen display face. Four tiers: statics
  (Regular/Bold/Italic/BoldItalic, 16-way calt texture cycling),
  a lean variable font (wght + opsz), VF Full (4-way real texture
  cycling in a variable font), and web woff2 of everything.
  Full GF Latin Core coverage: EN/RO/VI plus CE, Baltic, Turkic,
  Welsh, Icelandic.
